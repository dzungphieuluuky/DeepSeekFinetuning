# DeepSeekFinetuning

This repository is used to finetune the DeepSeek OCR model on a handwritten Vietnamese dataset, specifically the UIT_HWDB_line dataset.

## Overview

- **Model:** DeepSeek OCR
- **Dataset:** UIT_HWDB_line (handwritten Vietnamese text)

## Usage

1. Clone this repository.
2. Prepare the UIT_HWDB_line dataset.
3. Follow the provided scripts and instructions to finetune the DeepSeek OCR model on your data.

## Goals

- Improve OCR accuracy for handwritten Vietnamese text.
- Provide scripts and documentation for reproducible finetuning.

## Requirements

- Python 3.x (Running on Google Colab)
- DeepSeek OCR dependencies (see Unsloth tutorial notebook)
